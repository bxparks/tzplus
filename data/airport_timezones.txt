# Manually maintained mapping of IATA or ICAO airport codes and timezones such
# that:
#
# 1) Every timezone is associated with at least one airport code.
# 2) An airport code is associated with only a single timezone.
# 3) Some timezones may be associated with multiple airports.
#
# Most of the airport names and codes were extracted from the corresponding
# wikipedia page, using the following guidelines:
#
# 1) Wikipedia articles often embed non-ASCII characters with accents in their
# English names of the airport. All characters were converted into the nearest
# corresponding ASCII character.
# 2) Use the well-known 3-letter IATA code if available.
# 3) Use the 4-letter ICAO code if IATA code is not assigned.
# (e.g. BGDH at Danmarkshavn Airport, Greenland).
# 4) Use the sentinel "0000" if no airport exists near the town/region. This
# makes it impossible to select one of these timezones using an airport code.
# Fortunately, most of these are small villages or towns which chose to use a
# different DST rule than the rest of the larger region for various regions.
